[AWB Editor](https://edi.awbeditor.com/index?utm_source=chatgpt.com) is a specialized application for **air cargo freight forwarders, airlines, cargo agents, and logistics companies**. Its primary purpose is to create, print, manage, and electronically transmit air cargo shipping documents according to IATA standards. ([edi.awbeditor.com][1])

### Main functionality

#### 1. Create Air Waybills (AWB)

* Create IATA-compliant Master Air Waybills (MAWB)
* Create House Air Waybills (HAWB)
* Print on:

  * Blank A4/Letter paper
  * Pre-printed AWB stationery
* Export AWBs as PDF. ([edi.awbeditor.com][1])

#### 2. Electronic AWB (eAWB / FWB)

One of its biggest features is electronic messaging.

It can:

* Create **FWB** Cargo-IMP messages
* Transmit **eAWB**
* Receive airline responses:

  * FMA (Accepted)
  * FNA (Rejected/Error)
* Support electronic document workflows instead of paper AWBs. ([web.awbeditor.com][2])

#### 3. Cargo-IMP Messaging

Supports several air cargo message types, including:

* FWB
* FHL
* XFWB
* XFZB
* Cargo-IMP transmission to connected airlines. ([edi.awbeditor.com][1])

#### 4. Dangerous Goods Documentation

Create and print:

* Shipper's Declaration for Dangerous Goods (DGD)
* IATA-compliant dangerous goods forms. ([edi.awbeditor.com][1])

#### 5. Cargo Labels

Generate:

* Barcoded cargo labels
* Shipment labels
* Package labels. ([edi.awbeditor.com][1])

#### 6. Cargo Manifests

Produce shipment manifests for airline cargo operations. ([edi.awbeditor.com][1])

#### 7. Bills of Lading

Supports creation and printing of Bills of Lading (B/L). ([edi.awbeditor.com][1])

#### 8. Contact Database

Store:

* Shippers
* Consignees
* Agents
* Airlines
* Automatically reuse contact information in new documents. ([awbeditor.com][3])

#### 9. Templates

Create reusable templates for frequently shipped cargo to reduce repetitive data entry. ([awbeditor.com][3])

#### 10. AWB Number Management

* Manage airline AWB stock
* Automatically assign AWB numbers
* Generate document numbers for other forms. ([awbeditor.com][3])

#### 11. Reports

Export shipment and invoice data to Excel for operational reporting and analysis. ([awbeditor.com][4])

#### 12. Shared Database

Supports:

* Multi-user access
* Cloud database
* Local shared databases
* User login and activity tracking. ([awbeditor.com][5])

## Typical workflow

A freight forwarder using AWB Editor would typically:

1. Create a MAWB or HAWB.
2. Select shipper and consignee from saved contacts.
3. Enter cargo details.
4. Print the AWB or save it as PDF.
5. Transmit the FWB/eAWB electronically to the airline.
6. Receive airline acceptance (FMA) or rejection (FNA).
7. Print labels and manifests.
8. Export shipment reports. ([web.awbeditor.com][2])
